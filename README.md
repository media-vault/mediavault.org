# mediavault.org

The official website and documentation for The MediaVault Project.

The goal of this project is to keep all information related to The MediaVault Project all in one place for ease of access.

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## Installation

```console
npm install
```

## Local Development

To run locally, please use:

```console
npm start
```
